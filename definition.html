In computer programming the programming languages are used to apply algorithms. Most programming languages consist of the instructions on the computer. Many programming languages are written in an imperative style (I.e., as a set of operations to carry out) Whereas other languages use the declarative form (i.e. you select the desired result). not how to achieve it). The programming language definition is generally divided into two components of the syntax (form) and the semantic (meaning). A specification document describes such languages (for example, an ISO standard specifies the programming language for C) Whereas other languages (such as Perl) have a dominant implementation which is considered the language of reference.
